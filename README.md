## Frequently Asked Questions

**Why is my location not loading?**
Make sure atmos has permission to access your location. Go to **Settings → Privacy & Security → Location Services → atmos** and select *While Using the App*.

**Why does the weather show mock data?**
atmos uses Apple WeatherKit to fetch real weather. If you see generic data, check your internet connection and location permissions.

**Does atmos collect my data?**
atmos does not collect, store, or share any personal data. Location is used only to fetch local weather and is never transmitted to our servers.

**Does atmos require an account?**
No account or sign-in is required.
